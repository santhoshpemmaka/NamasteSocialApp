# Namaste Social Media App

Tech-
- ReactJs
- Mockbee API
- SCSS
- Redux ToolKit

Functionalities-

- Compose Post, Posts feed, Like Post
- Bookmark Post, delete Post

- Follow and unfollow User

- Show followers and following list
